# Local Plana Model

Put your local Plana Live2D files here. This directory is ignored by git.

Recommended layout:

```text
public/models/plana-local/
  plana.model3.json
  textures/
  motions/
  expressions/
```

Then open the viewer with:

```text
http://127.0.0.1:5173/?model=/models/plana-local/plana.model3.json
```

Do not commit copyrighted game assets to this repository.
